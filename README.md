# GTAA Invoice Extraction & Processing Automation

## Overview
This project automates the invoice processing workflow using Microsoft Power Automate and AI Builder.

The automation detects incoming invoice emails, extracts invoice data from PDF attachments, stores the extracted information into Excel, processes the data according to business rules, and sends the final processed Excel file back via email.

---

## Features
- Automated email trigger
- Invoice PDF extraction
- AI Builder invoice processing
- Excel data storage
- Business rule validation
- Automated email response
- Reduced manual work
- Improved accuracy

---

## Workflow

1. Monitor mailbox for emails with subject "GTAA Invoice"
2. Trigger flow automatically
3. Download invoice PDF attachment
4. Extract invoice details using AI Builder
5. Store extracted data in Excel
6. Process data as per client requirement
7. Generate final Excel report
8. Send processed report through email

---

## Extracted Fields
- Vendor Name
- Invoice Number
- Invoice Date
- Due Date
- Invoice Amount
- Tax Amount
- Currency
